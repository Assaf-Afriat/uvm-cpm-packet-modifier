DUT Overview

The CPM (Configurable Packet Modifier) is a packet processing unit that accepts packets via a stream interface, modifies payloads based on configurable operation modes, and outputs processed packets.

Single-Beat Packets

ID (4b), Opcode (4b), Payload (16b)

4 Operation Modes

PASS, XOR, ADD, ROT

Configuration & Counters

Operation Modes

Mode Value Transformation Latency
PASS 0 payload_out = payload_in 0 cycles
XOR 1 payload_out = payload_in ^ MASK 1 cycle
ADD 2 payload_out = payload_in + ADD_CONST 2 cycles
ROT 3 payload_out = rotate_left(payload_in, 4) 1 cycle

User Guide

Quick start guide and command reference

Prerequisites

  • QuestaSim 2025.1 or later
  • Python 3.x
  • UVM 1.1d library

Running Tests

# Navigate to project directory
cd "Verification/Final Project"

# Run smoke test
python scripts/Run/run.py --test CpmSmokeTest --timeout 60

# Run main test with coverage
python scripts/Run/run.py --test CpmMainTest --coverage-report

# Run with GUI mode
python scripts/Run/run.py --test CpmMainTest --gui
